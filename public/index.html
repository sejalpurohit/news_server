GET /api/articles

Description:Retrieves a list of all Articles.

Acceptable Queries:

  • ?sort_by – which sorts the articles by any valid column
  • order= – order, which can be set to ASC or DESC
  • ?topic – topic, which can be set to any existing topic

Example Request:

GET http://localhost:9000/api/articles
GET http://localhost:9000/api/articles?sort_by=article_id&order=desc
GET http://localhost:9000/api/articles/?topic=cats

Example Response:

POST/api/articles/:article_id/comments

Description: Post comments to a specific article by Id

Example Request:

POST http://localhost:9000/api/articles/1/comments

Request Body (JSON):


{
username: "butter_bridge",
body: "Its a beautiful day let go fow a walk",
};

Example Response:


comment: {
comment_id: 19,
body: 'Its a beautiful day let go fow a walk',
votes: 0,
created_at: '2025-10-28T21:27:50.533Z',
article_id: 1,
author: 'butter_bridge'
}

PATCH /api/articles/:article_id

Description: Update existing article. Update number of votes on an existing Id.

Example Request:

PATCH http://localhost:9000/api/articles

Request Body (JSON):


votes = { inc_votes: 50 }

Example Response:


article: {
article_id: 7,
title: 'Z',
body: 'I was hungry.',
created_at: '2020-01-07T14:08:00.000Z',
votes: 50,
article_img_url: 'https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700',
topic: 'mitch',
author: 'icellusedkars'
}

POST /api/articles

Description: Post new article.

Example Request:

POST http://localhost:9000/api/articles

Request Body (JSON):


article = {
title: "black cat",
body: "black cat is cute",
article_img_url: "https://dummyimage",
topic: "mitch",
author: "butter_bridge",
}

Example Response:


{
article_id: 14,
title: 'black cat',
body: 'black cat is cute',
article_img_url: 'https://dummyimage',
votes: 0,
topic: 'mitch',
author: 'butter_bridge',
created_at: '2025-10-28T21:40:21.533Z',
comment_count: 0
}

{}
